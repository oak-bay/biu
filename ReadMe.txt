biu 项目概述
	基于命令行的数据处理程序.
========================================================================
biu 
	-h 打印帮助命令.
	-h=<block_name> 打印指定模块的帮助命令.
	-b=<block_name> prop1=value1 prop2=value2 ... 依次添加模块.
